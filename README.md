#### presentation
To 'view' the presentation, "pursue" the steps below (needs node/nvm):
```shell
 $ cd revealJs_presentation
 $ nvm use
 $ npm install
 $ npm run build
 $ npm run start
 $ (xdg-)open http://localhost:8000
```
